Detailed Description

template<typename ConnectionHandler>
class cs_net::generic_server< ConnectionHandler >

Listening port, accepts client connections, instantiates connection handler.

Generic server listening port for incoming connections. Once connection encountered, accepts client connection. Instantiates connection handler for each client.
